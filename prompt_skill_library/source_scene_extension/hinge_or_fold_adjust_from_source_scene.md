# 原素材场景铰链/折叠调整

## 用途

用于把上传素材首帧延展成一个 5 秒竖屏写实商品短视频镜头: 在原素材场景内, 用手稳定商品固定部分, 再让可动部分围绕可见或可推断的铰链轴、折叠轴、转轴、支架轴完成一次开合或角度调整。该 skill 服务于商品带货分镜/prompt 生成, 重点是顺着素材首帧理解真实结构, 不重新发明产品形态, 不换地点, 不在同一个镜头里叠加多个功能演示。

本 skill 只能生成一个主动作: "围绕固定轴调整角度"。允许手指为了压稳或拨动边缘而有微小位置变化, 允许镜头轻微推近或侧移帮助看清铰链, 但不能同时完成拿起、翻面、拆开、插线、开机、屏幕亮起、logo 特写、收纳入包等第二主动作。

## 适用条件

- 素材首帧中能看到或合理推断固定轴/折叠轴/铰链方向, 例如支架、翻盖、折叠屏、带铰链设备、可调角度底座、翻折包装结构。
- 商品的固定部分和移动部分能被区分, 且固定部分在素材里有明确支撑关系, 例如底座贴桌面、背板靠支架、盒体放在台面上。
- 用户希望展示结构开合、角度变化、支撑稳定性或可调节功能, 并接受动作发生在原素材场景里。
- 动作幅度适合 5 秒完成, 例如从半开调到更开、从平放抬起一侧、从 45 度调到 70 度。
- 背景允许沿素材首帧自然补足边缘, 但必须保持同一桌面、同一房间、同一光线和同一商品身份。

## 禁用条件

- 用户要求同一镜头从素材首帧切到新地点, 例如首帧是办公桌上的铰链设备, 后面变成户外露营桌、健身房、车内、门店货架或纯色棚拍。
- 用户要求一个 5 秒镜头同时完成多个主动作, 例如打开支架后拿起设备旋转展示, 再按按钮亮屏, 再合上收纳。
- 素材中无法判断铰链轴或折叠方向, 但用户要求精确机械开合。
- 商品结构、logo、屏幕边框、转轴、铰链孔位、折痕线是核心卖点, 但用户要求大幅翻转、绕场移动或遮挡后重现。
- 手还没有接触固定部分或移动边缘, 商品可动部分已经自己开始转动。
- 用户要求把原商品改成另一种结构, 例如普通平板变成双折叠屏, 单轴支架变成多段机械臂, 无 logo 产品变成带新 logo 产品。
- 需要的是纯静态材质/logo 展示, 不需要开合动作; 此时应改用静态质感展示类 skill。

## 必填槽位

- `{source_scene_description}`: 对素材首帧空间的约束描述, 必须包含承托面、背景、光线和可扩展边界。例如: "深色办公桌面, 背后是模糊的显示器底座和一段灰色墙面, 左侧有暖色台灯反光, 光线从画面左上方来"。
- `{product_description}`: 主商品首帧可见身份, 必须写清固定部分、移动部分和关键识别点。例如: "黑色带 logo 铰链设备, 下方是矩形底座, 上方薄板通过银色铰链与底座连接, 正面有白色小 logo"。
- `{product_location}`: 商品位置和固定支撑关系。例如: "画面中央的深色桌面上, 矩形底座完整贴住桌面"。
- `{hinge_axis}`: 转动轴的位置和方向, 必须具体。例如: "底座后沿的水平银色铰链轴, 从画面左侧延伸到右侧"。
- `{stabilizing_hand}`: 稳定固定部分的手及接触方式。例如: "左手从画面左下方进入, 手掌压住底座左前角"。
- `{moving_hand}`: 拨动/拉动可动部分的手及接触方式。例如: "右手从画面右侧进入, 拇指和食指捏住上方薄板右边缘"。
- `{target_angle}`: 目标角度或开合幅度, 不要夸张。例如: "约 70 度", "从半开调到接近直立但未超过 80 度"。
- `{end_state}`: 最后一帧状态, 必须说明固定部分、移动部分、接触关系和原场景仍可见。例如: "底座仍贴住桌面, 上方薄板停在约 70 度, 双手轻轻离开边缘, logo 和铰链仍清楚可见"。
- `{single_action}`: 若 composer 支持, 固定填为 "铰链/折叠角度调整"; 不要填入 "打开并旋转展示再合上"。

## Prompt 块结构

生成 prompt 时按以下结构组织, 不要写成计划说明, 要直接给视频生成模型可执行的画面指令。填槽后总长度建议 900-1400 中文字符。

```text
竖屏 9:16, 写实商品带货短视频, 时长 5 秒, 第一帧必须就是参考图里的真实画面。不要重建成新场景, 不要切镜, 不要换地点, 不要新增第二个同款商品。画面环境必须顺着第一帧理解并自然延展: {source_scene_description}。主商品是{product_description}, 位于{product_location}; 商品颜色、比例、logo、屏幕边框、转轴、铰链孔位、折痕线、材质反光和首帧保持一致。

本镜头只做一个主动作: {single_action}, 也就是可动部分围绕{hinge_axis}缓慢调整到约{target_angle}。允许镜头轻微推近或侧移, 允许手指为了压稳和拨动有微调, 但不能同时做拿起、翻面、拆装、按键开机、屏幕亮起、插线、收纳、跨地点展示。必须先由{stabilizing_hand}接触并稳定固定部分, 再由{moving_hand}接触可动部分, 之后可动部分才能开始转动。

0.0-0.7 秒: 完全承接参考图首帧, 商品仍在{product_location}, 固定部分保持原支撑关系, 背景物、桌面纹理、光线方向和阴影都沿用素材。镜头稳定, 只允许非常轻微的推进, 让观众确认这是素材里的同一件商品和同一个铰链结构。

0.7-1.5 秒: {stabilizing_hand}先进入并压稳固定部分, 接触点出现真实遮挡和贴合阴影; {moving_hand}随后靠近可动部分边缘, 手指先接触再发力。此阶段可动部分仍保持首帧角度, 不能提前自动转动, 固定底座不能滑动或翘起。

1.5-3.1 秒: 可动部分只围绕{hinge_axis}缓慢转动, 从首帧角度调整到约{target_angle}。固定部分始终贴住承托面或保持原有支撑, 不能整件商品跳起、漂浮、穿过桌面。铰链轴位置稳定, 两侧结构连接合理, logo、边框、折痕、螺丝孔、材质反光不漂移、不变形、不重新设计。

3.1-4.1 秒: 调整动作完成后, 手的力度逐渐减小, 商品停在稳定角度。可动部分不能回弹乱晃, 不能继续完成第二次开合; 背景只允许同一素材场景内轻微虚化或边缘自然延展, 不变成新地点或棚拍。

4.1-5.0 秒: 镜头轻微侧移或推近展示结构关系, 最后一帧是{end_state}。画面中只有一个主商品, 固定部分、可动部分、铰链轴和接触关系清楚, 原素材场景的承托面、背景和光线仍可辨。
```

## 正例

### 正例 1: 带 logo 铰链设备示例

```text
竖屏 9:16, 写实商品带货短视频, 时长 5 秒, 第一帧必须就是参考图里的真实画面。不要重建成新场景, 不要切镜, 不要换地点, 不要新增第二个同款设备。画面环境必须顺着第一帧理解并自然延展: 深色办公桌面, 背后是虚化的显示器底座和一段灰色墙面, 画面左后方有暖色台灯反光, 光线从左上方落在桌面和商品边缘。主商品是黑色带 logo 铰链设备, 下方是矩形底座, 上方薄板通过银色水平铰链与底座连接, 薄板正面右下角有白色小 logo, 位于画面中央的深色桌面上, 矩形底座完整贴住桌面; 商品比例、黑色磨砂材质、白色 logo、银色铰链轴和首帧保持一致。

本镜头只做一个主动作: 铰链角度调整, 也就是上方薄板围绕底座后沿的水平银色铰链轴, 从首帧半开状态缓慢调整到约 70 度。允许镜头轻微推近, 允许手指为了压稳和拨动有微调, 但不能同时把设备拿起来、翻到背面、按键开机、让屏幕亮起、插线、收纳或切到新办公桌。必须先由左手从画面左下方进入并压稳底座左前角, 再由右手从画面右侧进入捏住上方薄板右边缘, 之后薄板才能开始转动。

0.0-0.7 秒: 完全承接参考图首帧, 设备仍在画面中央的深色桌面上, 底座贴住桌面, 上方薄板保持原半开角度, 显示器底座、灰色墙面、台灯暖色反光和桌面纹理位置不变。镜头稳定, 只做非常轻微的推进, 让观众确认这是素材里的同一件黑色铰链设备。

0.7-1.5 秒: 左手从画面左下方进入, 手掌轻压底座左前角, 接触处出现真实遮挡和贴合阴影; 右手从画面右侧进入, 拇指和食指靠近并捏住上方薄板右边缘。此阶段薄板仍保持首帧角度, 不能提前自动转动, 底座不能滑动或翘起。

1.5-3.1 秒: 右手缓慢抬动薄板边缘, 薄板只围绕底座后沿的水平银色铰链轴转动, 从半开状态调整到约 70 度。底座始终完整贴住桌面, 整件设备不能跳起、漂浮或穿过桌面。银色铰链轴位置稳定, 两侧连接关系合理, 白色小 logo、黑色边框、螺丝孔和磨砂反光不漂移、不变形、不重新设计。

3.1-4.1 秒: 调整完成后, 右手力度逐渐减小, 左手仍轻压底座让设备停在稳定角度。薄板不能继续第二次开合, 不能回弹乱晃; 背景只允许轻微虚化, 仍然是原来的深色办公桌面、显示器底座和灰色墙面。

4.1-5.0 秒: 镜头轻微向右侧移并推近一点, 最后一帧是底座仍贴住桌面, 上方薄板停在约 70 度, 双手轻轻离开边缘但仍在画面近处, 白色 logo 和银色铰链清楚可见。画面中只有一个主设备, 原素材场景的桌面、背景和暖色光线仍可辨。
```

### 正例 2: 折叠支架示例

```text
竖屏 9:16, 写实商品带货短视频, 时长 5 秒, 第一帧必须就是参考图里的真实画面。不要重建成新场景, 不要切镜, 不要换地点, 不要新增第二个支架。画面环境必须顺着第一帧理解并自然延展: 浅木纹桌面, 后方是虚化的白色书架下沿和一小段绿色植物, 右侧边缘有柔和窗光, 整体是室内自然光。主商品是银灰色折叠支架, 底部平板贴在桌面上, 后方支撑臂通过圆形转轴连接底板, 支撑臂上有一条细长凹槽和小型品牌标识, 位于画面下半部中央的浅木纹桌面上; 商品比例、金属颜色、转轴圆孔、凹槽和品牌标识保持首帧一致。

本镜头只做一个主动作: 折叠支架角度调整, 也就是后方支撑臂围绕底板后端的圆形转轴, 从较低角度缓慢抬到约 60 度。允许镜头轻微推近, 允许手指微调压稳位置, 但不能同时放上手机、拿起支架、旋转支架、折回收纳、展示包装或切换到旅行场景。必须先由左手从画面左侧进入压住底部平板前沿, 再由右手从画面右上方进入捏住支撑臂上端, 之后支撑臂才能开始转动。

0.0-0.7 秒: 完全承接参考图首帧, 银灰色折叠支架仍在浅木纹桌面上, 底部平板贴住桌面, 后方支撑臂保持原本较低角度, 书架下沿、绿色植物和窗光方向不变。镜头稳定, 只允许轻微推进。

0.7-1.5 秒: 左手从画面左侧进入, 食指和中指压住底部平板前沿, 接触处出现自然阴影; 右手从画面右上方进入, 拇指和食指捏住支撑臂上端。此阶段支撑臂仍保持首帧角度, 不能自己抬起, 底板不能滑动。

1.5-3.1 秒: 右手缓慢抬起支撑臂上端, 支撑臂只围绕底板后端的圆形转轴转动, 从较低角度调整到约 60 度。底部平板始终贴住浅木纹桌面, 不能整件支架跳起或穿过桌面。圆形转轴、细长凹槽、小型品牌标识和银灰金属反光保持稳定, 不多出新的支臂, 不把单轴结构改成多段机械臂。

3.1-4.1 秒: 支撑臂到达约 60 度后, 右手慢慢放松, 左手仍轻压底板, 支架停在稳定角度。支撑臂不能继续折回, 不能突然放上手机; 背景只做同一室内场景的轻微虚化, 仍然能看见浅木纹桌面和白色书架边缘。

4.1-5.0 秒: 镜头轻微推近支架侧面, 最后一帧是底部平板仍贴住桌面, 支撑臂稳定停在约 60 度, 双手刚离开接触点, 圆形转轴、细长凹槽和品牌标识清楚可见。画面中只有一个主支架, 原素材场景的桌面、书架和自然窗光仍可辨。
```

## 反例

- 反例: "第一帧是办公桌上的黑色铰链设备, 1 秒后镜头切到户外露营桌, 手在那里打开支架。" 失败原因: 素材首帧 + 新地点冲突; 本 skill 只能扩展原素材场景, 不能同一镜头跨地点。
- 反例: "手打开支架, 放上手机, 点亮屏幕, 旋转展示背面 logo, 再把支架折回。" 失败原因: 多动作塞进一个镜头; 5 秒内只允许一次铰链/折叠角度调整。
- 反例: "薄板自己先抬起到 70 度, 右手随后出现扶住边缘。" 失败原因: 手未接触先移动; 违反接触关系和机械受力, 应标记 premature_motion 或 bad_physics。
- 反例: "带 logo 铰链设备在开合后, logo 从右下角变到中央, 铰链从银色水平轴变成两个黑色圆扣。" 失败原因: logo/结构被大幅重绘; 商品身份和机械结构漂移, 应标记 logo_changed 或 shape_changed。
- 反例: "底座在桌面上固定不住, 调整时整件商品悬浮起来, 但 prompt 仍说是开合。" 失败原因: 固定部分没有稳定支撑, 物理错误, 应标记 bad_physics。
- 反例: "原素材是浅木纹桌面, prompt 写成白底棚拍, 只保留支架。" 失败原因: 抹除了素材首帧空间, 属于 scene_conflict。
- 反例: "铰链设备打开时长出第二段支臂, 从单轴支架变成多轴机械臂。" 失败原因: 结构被重绘, 不再是首帧商品, 应标记 shape_changed。
- 反例: "5 秒内手只是摸了一下铰链, 角度没有变化。" 失败原因: 主动作未完成, 应标记 action_not_completed。

## 失败标签

- `scene_conflict`: 背景、地点、桌面、承托面、光线或空间边界与素材首帧冲突。
- `bad_physics`: 固定部分滑动、翘起、漂浮, 可动部分穿插桌面或整件商品瞬移。
- `premature_motion`: 手尚未接触固定部分/可动部分, 铰链或折叠结构已经开始转动。
- `shape_changed`: 固定部分、移动部分、转轴、折痕线、边框、支臂数量或整体结构被明显改写。
- `logo_changed`: logo、文字、品牌标识的位置、大小、颜色或内容明显漂移。
- `action_not_completed`: 角度没有调整到 `{target_angle}`, 或动作停在未完成状态。
- `extra_product`: 生成第二个同款设备, 或原位置残留一个复制品。
- `action_overload`: 同镜头包含调整角度之外的拿起、翻面、开机、插线、放入手机、收纳、跨场景展示等主动作。

## 校验规则

- 首帧校验: prompt 必须明确 "第一帧就是参考图里的真实画面", 且 0.0-0.7 秒承接素材首帧。
- 场景校验: 所有时间块都必须留在 `{source_scene_description}` 的同一地点; 允许边缘自然延展, 不允许新地点、新桌面、新房间、户外或纯色棚拍替换原场景。
- 单动作校验: 5 秒内只有 "铰链/折叠角度调整" 一个主动作; 镜头移动、手指压稳、短暂停住只能作为辅助。
- 接触校验: `{stabilizing_hand}` 必须先稳定固定部分, `{moving_hand}` 必须先接触可动部分, 之后可动部分才围绕 `{hinge_axis}` 转动。
- 轴线校验: 可动部分只能围绕 `{hinge_axis}` 转动; 不能整件商品平移、翻滚、绕不存在的轴旋转。
- 固定部分校验: 底座、盒体或固定面必须保持原支撑关系, 不能滑动、翘起、漂浮或穿桌。
- 身份校验: `{product_description}` 中的 logo、文字、边框、铰链孔位、折痕线、材质反光和结构数量必须贯穿 0-5 秒。
- 终帧校验: 4.1-5.0 秒必须给出 `{end_state}`, 说明目标角度、固定部分状态、可动部分状态、手的接触/离开关系和原素材场景仍可见。

## 给 LLM 的决策提示

- 如果用户要求 "打开看看结构", 优先选择短幅度、清晰轴线的角度调整, 不要把商品拿起翻面。
- 如果素材里轴线不清楚, 但能看出一条合理折痕或转轴, 在 `{hinge_axis}` 中保守描述为 "可见折痕/后沿转轴", 并限制角度变化不要太大。
- 如果 logo、屏幕边框或铰链孔位是核心识别点, 缩短动作路径, 减少遮挡和大角度旋转, 避免 logo_changed 或 shape_changed。
- 如果用户同时要求打开、放手机、点亮屏幕、收纳, 本镜头只保留 "调整角度", 其余动作应拆成后续镜头。
- 如果素材首帧与用户指定新地点冲突, 本 skill 应拒绝跨地点改写, 改为 "在原素材场景中调整铰链/折叠角度" 或交给新场景重构策略。
- 如果固定部分在素材里没有可靠支撑, 不要强行生成复杂开合; 改用静态质感展示或请求更清晰素材。

## Prompt 模板

竖屏 9:16, 写实商品带货短视频, 时长 5 秒, 第一帧必须就是参考图里的真实画面。不要重建成新场景, 不要切镜, 不要换地点, 不要新增第二个同款商品。画面环境必须顺着第一帧理解并自然延展: {source_scene_description}。主商品是{product_description}, 位于{product_location}; 商品颜色、比例、logo、屏幕边框、转轴、铰链孔位、折痕线、材质反光和首帧保持一致。

本镜头只做一个主动作: {single_action}, 也就是可动部分围绕{hinge_axis}缓慢调整到约{target_angle}。允许镜头轻微推近或侧移, 允许手指为了压稳和拨动有微调, 但不能同时做拿起、翻面、拆装、按键开机、屏幕亮起、插线、收纳、跨地点展示。必须先由{stabilizing_hand}接触并稳定固定部分, 再由{moving_hand}接触可动部分, 之后可动部分才能开始转动。

0.0-0.7 秒: 完全承接参考图首帧, 商品仍在{product_location}, 固定部分保持原支撑关系, 背景物、桌面纹理、光线方向和阴影都沿用素材。镜头稳定, 只允许非常轻微的推进, 让观众确认这是素材里的同一件商品和同一个铰链结构。

0.7-1.5 秒: {stabilizing_hand}先进入并压稳固定部分, 接触点出现真实遮挡和贴合阴影; {moving_hand}随后靠近可动部分边缘, 手指先接触再发力。此阶段可动部分仍保持首帧角度, 不能提前自动转动, 固定底座不能滑动或翘起。
