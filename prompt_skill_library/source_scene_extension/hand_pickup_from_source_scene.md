# 原素材场景手部拿起

## 用途

用于把上传素材的首帧延展成一个 5 秒竖屏写实商品短视频镜头: 商品从素材里原本的位置被一只手真实接触、握住、拿起或轻微移开。该 skill 服务于商品带货分镜/prompt 生成, 重点不是重新设计场景, 而是让模型严格顺着素材首帧理解空间、光线、桌面、背景、商品身份和接触关系。

本 skill 只能生成一个主动作: "手拿起商品"。允许有轻微镜头推进、轻微景深变化、手指微调握姿, 但这些都必须服务于同一次拿起动作, 不能把展示、旋转、倒水、开盖、饮用、放进包里等动作塞进同一个 5 秒镜头。

## 适用条件

- 素材首帧中主商品清楚可见, 商品边缘、底部支撑点、桌面或承托面能被合理判断。
- 需要从原素材场景自然延展, 例如桌面上的杯子被拿起、架子上的小设备被手拿离原位、包装盒被从台面上轻轻拿起。
- 商品身份主要依赖首帧保真, 包括颜色、比例、logo、标签、透明材质、结构线、盖子位置、把手位置等。
- 动作目标可以在 5 秒内完成, 且只需要一只手或双手中的一只主手完成拿起。
- 背景允许小幅扩展首帧未显示的边缘区域, 但扩展出的内容必须像同一桌面、同一房间、同一光线下自然延伸。

## 禁用条件

- 用户要求同一镜头从素材首帧切到新地点, 例如首帧是厨房桌面, 后面变成户外咖啡馆、车内、浴室、办公室货架或纯色棚拍。
- 用户要求一个 5 秒镜头完成多个主动作, 例如拿起后旋转展示再打开盖子再倒水, 或拿起后走到窗边再饮用。
- 手还没有接触商品就让商品移动、悬浮、滑走、自动升起。
- 素材中商品被严重遮挡, 无法判断可抓取部位或底部支撑关系。
- 商品包含必须完全精确的 logo、二维码、密集文字、复杂结构, 且动作要求大幅转向或遮挡再重现。
- 用户要求重绘商品造型、替换包装、改变 logo、改变透明度、改变容量刻度、改成另一个型号。
- 拍摄意图是静态质感特写, 不需要手拿起; 此时应改用静态质感展示类 skill。

## 必填槽位

- `{source_scene_description}`: 对素材首帧的空间理解, 必须写清同一地点、承托面、背景元素、光线方向和可扩展边界。例如: "浅色厨房台面, 后方是模糊的水槽和白色墙砖, 左侧边缘有一小段砧板, 自然侧光从画面右上方进入"。
- `{product_description}`: 主商品的可见身份, 必须锚定首帧外观, 不新增型号特征。例如: "透明圆柱水杯, 直身杯壁, 银色杯盖, 杯身有一枚小白色 logo, 杯底略厚"。
- `{product_location}`: 商品首帧所在位置和支撑关系。例如: "画面下半部中央的浅色台面上, 杯底完整贴住台面"。
- `{hand_entry}`: 手从哪里进入及大致方向。例如: "画面右侧中部伸入, 手背朝向镜头, 指尖先靠近杯身右侧"。
- `{contact_part}`: 首次真实接触的商品部位。例如: "杯身中上段和杯盖下沿之间的右侧曲面"。
- `{lift_path}`: 拿起路径, 必须短、连续、符合物理。例如: "垂直抬高约 8 厘米后向画面右上方移动半个杯身宽度"。
- `{end_state}`: 最后一帧状态, 必须仍在原素材场景边界内。例如: "手握着水杯停在台面上方, 杯底离开原位置, 原台面和后方白色墙砖仍可见"。
- `{single_action}`: 若 composer 支持, 固定填为 "手拿起商品"; 不要填入 "拿起并旋转展示" 这类复合动作。

## Prompt 块结构

生成 prompt 时按以下结构组织, 不要写成计划说明, 要直接给视频生成模型可执行的画面指令。填槽后总长度建议 900-1400 中文字符。

```text
竖屏 9:16, 写实商品带货短视频, 时长 5 秒, 第一帧必须就是参考图里的真实画面。不要重建成新场景, 不要切镜, 不要换地点, 不要新增第二个同款商品。画面环境必须顺着第一帧理解并自然延展: {source_scene_description}。主商品是{product_description}, 位于{product_location}; 商品颜色、比例、logo、标签、透明度、边缘结构和首帧保持一致。

本镜头只做一个主动作: {single_action}。允许手指为了握稳而轻微调整, 允许镜头有很慢的推近或微弱手持呼吸感, 但不能同时做开盖、倒出内容物、饮用、拆装、旋转 360 度展示、走路、跨地点展示。手必须先接触{contact_part}, 商品才能开始移动; 商品不能隔空被操控、不能漂浮、不能穿过桌面或背景物。

0.0-0.7 秒: 完全承接参考图首帧, 商品仍在{product_location}, 桌面/承托面和周围物体位置不变。镜头稳定, 光线、阴影和反射沿用素材, 只允许极轻微推进, 让观众确认这就是素材里的同一件商品。

0.7-1.5 秒: 一只自然比例的手从{hand_entry}进入画面, 指尖先靠近但不推动商品。手的遮挡要合理, 手指在{contact_part}形成清楚接触, 接触点出现轻微压迫、反光变化或阴影贴合。此阶段商品仍保持原位, 不能提前滑动或抬起。

1.5-2.8 秒: 手指收拢并握稳后, 商品沿{lift_path}被连续拿起。底部先离开承托面, 原位置留下正常阴影变化; 拿起速度平稳, 没有跳帧、瞬移、抖动变形。商品整体刚体运动, logo、杯壁、盖子、标签、结构线不被大幅重绘。

2.8-3.9 秒: 手继续保持接触和承重, 商品停在同一素材场景内的近处位置。背景只做同一地点的轻微虚化或边缘延展, 不能变成新房间、新桌面、户外、车内或棚拍。商品朝向可以有小幅自然倾斜, 但不要变成完整旋转展示。

3.9-5.0 秒: 动作完成并稳定住, 最后一帧是{end_state}。画面中只有一个主商品, 手与商品接触关系清楚, 商品仍保留首帧外观和关键识别点, 原素材场景的台面、背景和光线仍然可辨。
```

## 正例

### 正例 1: 透明水杯示例

```text
竖屏 9:16, 写实商品带货短视频, 时长 5 秒, 第一帧必须就是参考图里的真实画面。不要重建成新场景, 不要切镜, 不要换地点, 不要新增第二个同款商品。画面环境必须顺着第一帧理解并自然延展: 浅色厨房台面, 后方是模糊的白色墙砖和一段不锈钢水槽边缘, 画面左后方有虚化的木质砧板, 自然侧光从画面右上方照进来, 台面反光柔和。主商品是透明圆柱水杯, 直身杯壁, 银色杯盖, 杯身正面有一枚小白色 logo, 杯底略厚且有轻微折射, 位于画面下半部中央的浅色台面上, 杯底完整贴住台面; 商品颜色、比例、logo、透明度、杯盖高度和首帧保持一致。

本镜头只做一个主动作: 手拿起水杯。允许手指为了握稳而轻微调整, 允许镜头有很慢的推近, 但不能同时做打开杯盖、倒水、喝水、走到窗边、旋转 360 度展示或切换到新的使用场景。手必须先接触杯身中上段和杯盖下沿之间的右侧曲面, 水杯才能开始移动; 水杯不能隔空被操控、不能漂浮、不能穿过台面。

0.0-0.7 秒: 完全承接参考图首帧, 水杯仍在画面下半部中央的浅色台面上, 杯底阴影和台面反射保持稳定, 后方白色墙砖、水槽边缘和砧板位置不变。镜头稳定, 只做极轻微向前推进, 让观众确认这是素材里的同一只透明水杯。

0.7-1.5 秒: 一只自然比例的右手从画面右侧中部伸入, 手背略朝向镜头, 指尖先靠近杯身右侧但不推动杯子。拇指和食指在杯身中上段形成清楚接触, 杯壁反光被手指局部遮住, 接触阴影贴合真实。此阶段水杯仍保持原位, 不能提前滑动或抬起。

1.5-2.8 秒: 手指收拢并握稳后, 水杯垂直抬高约 8 厘米, 再向画面右上方移动半个杯身宽度。杯底先离开台面, 原位置的阴影自然减弱; 水杯整体保持刚体运动, 透明杯壁、银色杯盖、小白色 logo 和杯底折射不被大幅重绘, 杯身不拉长、不变扁、不多出第二个把手。

2.8-3.9 秒: 右手继续握住水杯并承重, 水杯停在台面上方的近处位置, 背景仍是同一厨房台面和白色墙砖, 只允许轻微虚化。水杯可以有很小的自然倾斜, 但不要变成完整旋转展示, 不要打开杯盖, 不要让液体出现。

3.9-5.0 秒: 动作完成并稳定住, 最后一帧是右手握着透明水杯停在台面上方, 杯底离开原位置, 原浅色台面、后方白色墙砖和水槽边缘仍可见。画面中只有一个主水杯, 手与杯身接触关系清楚, 水杯外观和 logo 仍然延续首帧。
```

### 正例 2: 小型包装盒示例

```text
竖屏 9:16, 写实商品带货短视频, 时长 5 秒, 第一帧必须就是参考图里的真实画面。不要重建成新场景, 不要切镜, 不要换地点, 不要新增第二个包装盒。画面环境必须顺着第一帧理解并自然延展: 深灰色办公桌面, 后方是虚化的笔记本电脑下沿和一盏暖色台灯反光, 画面右侧边缘有一小段黑色鼠标垫, 室内暖光从左上方落在桌面。主商品是白色长方体电子配件包装盒, 正面有深色品牌 logo 和一行小字, 盒身棱角清晰, 位于画面中央偏左的深灰色桌面上, 底面贴住桌面; 商品比例、白色纸质纹理、logo 位置和首帧保持一致。

本镜头只做一个主动作: 手拿起包装盒。允许手指为夹住盒身做轻微调整, 允许镜头有很慢的侧向微移, 但不能同时撕开封条、打开盒盖、取出配件、把盒子放进包里或切到棚拍白底。手必须先接触包装盒右侧边缘和上表面交界处, 包装盒才能开始移动; 包装盒不能自动滑动、不能变成软袋、不能穿过桌面。

0.0-0.7 秒: 完全承接参考图首帧, 包装盒仍在画面中央偏左的深灰色桌面上, 桌面纹理、电脑下沿、台灯暖色反光和鼠标垫边缘位置不变。镜头稳定, 光线沿用素材, 只允许微弱推进。

0.7-1.5 秒: 一只左手从画面左下方进入, 手心朝向桌面, 拇指靠近包装盒正面左侧, 食指和中指靠近盒子上表面。手指先形成真实接触, 盒子边缘被手指局部遮挡, 接触处有贴合阴影。此阶段包装盒仍不移动。

1.5-2.8 秒: 左手夹稳包装盒后, 沿着垂直抬高约 5 厘米再向画面左上方移动三分之一个盒身宽度的路径拿起。盒底离开桌面时, 原位置阴影自然变淡; 包装盒保持硬质长方体结构, 正面 logo 和小字位置不漂移, 盒角不塌陷, 不出现第二个盒子。

2.8-3.9 秒: 手继续夹住包装盒, 让盒子停在同一办公桌场景内的近处位置。背景可以轻微虚化, 但仍能看出是原来的深灰桌面、电脑下沿和暖色台灯反光, 不能变成货架、快递拆箱桌或纯色影棚。包装盒只允许有小幅自然倾斜, 正面仍朝向镜头, 不能转成背面说明书展示。

3.9-5.0 秒: 动作完成并稳定住, 最后一帧是左手拿着白色包装盒停在桌面上方, 原桌面、电脑下沿、鼠标垫边缘和暖色反光仍可见。画面中只有一个主包装盒, 手与盒身接触关系清楚, 盒身 logo、文字区、棱角和纸质纹理仍延续首帧。
```

## 反例

- 反例: "第一帧是厨房台面上的透明水杯, 1 秒后手拿起水杯走到户外阳台, 背景变成蓝天和植物。" 失败原因: 素材首帧 + 新地点冲突; source_scene_extension 只能在同一素材场景内扩展, 不能同镜头跨地点。
- 反例: "手拿起水杯, 旋转展示 logo, 打开杯盖, 倒出柠檬水, 最后喝一口。" 失败原因: 多动作塞进一个 5 秒镜头; 本 skill 只允许一个主动作, 其余动作会导致物理、结构和身份保真下降。
- 反例: "杯子先自己从桌面升起, 手随后从右侧出现握住。" 失败原因: 手未接触先移动; 接触关系违反真实物理, 应标记 premature_motion 或 bad_physics。
- 反例: "手拿起带 logo 包装盒后, logo 变大变成金色, 包装盒由方盒变成圆角瓶身。" 失败原因: logo/结构被大幅重绘; 商品身份漂移, 应标记 identity_drift。
- 反例: "手从桌面下方穿过杯底拿起杯子, 杯底没有阴影变化。" 失败原因: 手和商品/桌面穿插, 支撑关系错误, 应标记 bad_physics。
- 反例: "原素材是木桌, prompt 写成白色无影棚拍背景, 只保留商品。" 失败原因: 抹除了素材场景, 属于 scene_conflict。
- 反例: "拿起商品时画面里又生成一个同款商品留在原位。" 失败原因: 主商品复制, 应标记 extra_product。
- 反例: "5 秒内只有镜头轻微推近, 手没有进入, 商品没有被拿起。" 失败原因: 动作没有执行, 对拿起 skill 来说过于静态, 应标记 too_static。

## 失败标签

- `scene_conflict`: 背景、地点、承托面、光线或空间边界与素材首帧冲突。
- `bad_physics`: 手、商品、桌面出现穿插、漂浮、瞬移、隔空控制、阴影不随支撑关系变化。
- `premature_motion`: 手尚未接触商品, 商品已经开始移动。
- `identity_drift`: 商品颜色、比例、材质、logo、文字、透明度或结构被明显重绘。
- `extra_product`: 生成第二个主商品, 或原位置残留一个复制品。
- `too_static`: 应该拿起但商品未离开原位置, 或动作只剩镜头运动。
- `action_overload`: 同一个 5 秒镜头包含拿起之外的开盖、倒水、饮用、拆装、行走、跨场景展示等主动作。

## 校验规则

- 首帧校验: prompt 必须明确 "第一帧就是参考图里的真实画面", 且 0.0-0.7 秒承接素材首帧。
- 场景校验: 所有时间块都必须留在 `{source_scene_description}` 的同一地点; 允许边缘自然扩展, 不允许新地点、新桌面、新房间、户外或纯色棚拍替换原场景。
- 单动作校验: 5 秒内只有 "手拿起商品" 一个主动作; 镜头推进、手指调整和稳定停住只能作为辅助。
- 接触校验: 必须先出现手从 `{hand_entry}` 进入并接触 `{contact_part}`, 之后商品才沿 `{lift_path}` 移动。
- 物理校验: 商品底部离开承托面时, 阴影、遮挡、反射要随之变化; 商品不能穿过桌面、不能漂浮、不能瞬移。
- 身份校验: `{product_description}` 中的 logo、结构、材质、比例、颜色、透明度、标签位置必须贯穿 0-5 秒。
- 终帧校验: 4.0-5.0 秒必须给出 `{end_state}`, 说明手仍如何接触商品、商品在哪里、原素材场景哪些元素仍可见。

## 给 LLM 的决策提示

- 如果用户要求 "更有生活感", 优先用同一素材场景里的手部拿起、微弱镜头推进、自然阴影变化表达, 不要切换到生活化新地点。
- 如果用户要求 "展示细节/logo", 不要在拿起镜头中安排大幅旋转; 只允许小角度稳定露出, 或建议另起一个静态质感镜头。
- 如果素材首帧看不出手应该从哪里进, 选择遮挡最少、路径最短、不会穿过商品或桌面的方向作为 `{hand_entry}`。
- 如果商品有透明、反光或细小 logo, 拿起路径要短, 角度变化要小, 以降低 identity_drift。
- 如果用户同时提出拿起、打开、使用、倒出、放回, 只保留 "拿起" 作为本镜头主动作, 其余动作交给后续镜头。
- 如果素材首帧与用户指定新地点冲突, 本 skill 应拒绝跨地点改写, 改为 "在原素材场景中拿起" 或交给新场景重构策略。

## Prompt 模板

竖屏 9:16, 写实商品带货短视频, 时长 5 秒, 第一帧必须就是参考图里的真实画面。不要重建成新场景, 不要切镜, 不要换地点, 不要新增第二个同款商品。画面环境必须顺着第一帧理解并自然延展: {source_scene_description}。主商品是{product_description}, 位于{product_location}; 商品颜色、比例、logo、标签、透明度、边缘结构和首帧保持一致。

本镜头只做一个主动作: {single_action}。允许手指为了握稳而轻微调整, 允许镜头有很慢的推近或微弱手持呼吸感, 但不能同时做开盖、倒出内容物、饮用、拆装、旋转 360 度展示、走路、跨地点展示。手必须先接触{contact_part}, 商品才能开始移动; 商品不能隔空被操控、不能漂浮、不能穿过桌面或背景物。

0.0-0.7 秒: 完全承接参考图首帧, 商品仍在{product_location}, 桌面/承托面和周围物体位置不变。镜头稳定, 光线、阴影和反射沿用素材, 只允许极轻微推进, 让观众确认这就是素材里的同一件商品。

0.7-1.5 秒: 一只自然比例的手从{hand_entry}进入画面, 指尖先靠近但不推动商品。手的遮挡要合理, 手指在{contact_part}形成清楚接触, 接触点出现轻微压迫、反光变化或阴影贴合。此阶段商品仍保持原位, 不能提前滑动或抬起。

1.5-2.8 秒: 手指收拢并握稳后, 商品沿{lift_path}被连续拿起。底部先离开承托面, 原位置留下正常阴影变化; 拿起速度平稳, 没有跳帧、瞬移、抖动变形。商品整体刚体运动, logo、杯壁、盖子、标签、结构线不被大幅重绘。

2.8-3.9 秒: 手继续保持接触和承重, 商品停在同一素材场景内的近处位置。背景只做同一地点的轻微虚化或边缘延展, 不能变成新房间、新桌面、户外、车内或棚拍。商品朝向可以有小幅自然倾斜, 但不要变成完整旋转展示。

3.9-5.0 秒: 动作完成并稳定住, 最后一帧是{end_state}。画面中只有一个主商品, 手与商品接触关系清楚, 商品仍保留首帧外观和关键识别点, 原素材场景的台面、背景和光线仍然可辨。
