# 新场景文字复刻：背包侧袋通勤

## 用途
用于把素材中可被文字描述清楚的低 logo、低结构风险商品，放到一个全新的通勤场景里，生成 5 秒竖屏商品带货分镜。这个 skill 不要求视频模型严格继承素材首帧画面，也不要求把素材图作为第一帧；它要求 composer 先理解素材中的商品外观，再用自然语言在新地点复刻商品。

核心目标：
- 明确告诉视频模型：这是新的独立分镜、新地点、新时间，不从上一镜头或素材首帧继续。
- 让商品看起来像素材里的同一类商品：颜色、比例、轮廓、材质、主要结构要复刻；细小 logo、印刷字、复杂标识不强行精确。
- 只表现一个动作：背包在通勤移动，商品已经在侧袋中随背包轻微晃动并被展示。
- 防止模型把上一镜的桌面、手拿、包装盒、插入过程、室内背景塞进同一空间。

## 适用条件
满足以下条件时优先使用：
- 商品适合放在背包侧袋、外侧网袋、外侧弹力袋、杯袋或类似通勤收纳位中，例如水杯、保温杯、雨伞、小型喷雾瓶、小型户外瓶、小圆筒收纳罐。
- 商品主要身份可由文字复刻：整体颜色、材质、形状、尺寸比例、盖子、瓶身、把手、纹理、透明度、金属光泽、硅胶套、磨砂质感等可描述清楚。
- 素材图中有 logo 或文字，但不是购买决策的唯一识别点；可以写成“有一处小面积模糊品牌标识/浅色小标签”，不能要求一字不差。
- 商品结构简单，运动中不需要展开、折叠、旋转对齐、屏幕显示、精密机械咬合。
- 新分镜目标是“通勤携带状态”“户外随身状态”“从家到办公室路上的自然展示”，不是展示安装、使用教程或精密功能。

典型适合：
- 素材是米白色细长保温杯，黑色杯盖，杯身无大面积文字。新场景是地铁入口，杯子已经插在背包侧袋。
- 素材是半透明浅绿色水瓶，顶部有白色翻盖和挂环。新场景是校园林荫路，水瓶在黑色双肩包侧袋里。
- 素材是深蓝色折叠伞收纳筒，短圆柱形，外层织物有细竖纹。新场景是雨后写字楼门口，收纳筒在背包外侧口袋。

## 禁用条件
出现以下任一情况时不要使用本 skill：
- 高精度 logo、高精度文字、队徽、联名图案、复杂包装插画、屏幕 UI、二维码、条形码、证件文字必须完全一致。
- 商品身份强依赖独特结构，例如复杂鞋底、机械铰链、折叠支架、可变形玩具、精密按键阵列、透明内部零件。
- 用户要求使用素材图作为第一帧，同时又要求换成新地点；这是首帧素材和新地点冲突，必须改用原场景延展技能或取消素材首帧绑定。
- 商品尺寸明显不适合背包侧袋，例如笔记本电脑、大包装盒、台灯、锅具、大号收纳箱、需要双手托举的物体。
- 分镜同时包含“从桌上拿起、塞进侧袋、走进地铁、到达办公室、取出使用”等多个动作。
- 前几个镜头完全自由铺垫，与商品无关，只在最后一秒突然出现侧袋商品；这会导致商品和场景割裂。
- 素材有多张图且互相冲突，不能强行要求每张都出现在同一 5 秒镜头里。

## 必填槽位
- `source_material_observation`：composer 对素材的简短理解，不直接塞给视频模型当首帧命令。必须写商品来自素材、可见特征、不可精确保留项。例如“素材中商品是一只浅灰磨砂保温杯，直筒杯身，黑色平盖，杯口有窄银边；杯身一侧有小 logo，但不要求精确复现文字”。
- `new_scene_description`：新地点、新时间、新光线、新背景，不得沿用素材场景。例如“清晨写字楼外的人行道，玻璃幕墙反射柔和天光，背景有虚化的通勤人群和路边绿植”。
- `product_description`：面向视频模型的商品一句话描述，不能写“参考图里的那个东西”。
- `product_reconstructable_features`：可复刻特征列表，优先写大轮廓、颜色、材质、结构、比例、可见边缘、盖子/把手/透明液位等；低优先级写小 logo 或小标签的模糊存在。
- `product_location`：商品所在的具体承载位置，例如“深灰色双肩包右侧弹力网袋”“帆布通勤包左侧外袋”。必须是新场景内的空间位置。
- `subject_description`：背包和携带者的可见状态，例如“穿浅色风衣的通勤者背着深灰双肩包，从画面左侧经过”。
- `camera_position`：镜头位置和运动，例如“腰部高度、背包右后方 45 度近距离跟拍”。
- `single_action`：本镜头唯一动作。推荐值：“通勤者背着包向前走，侧袋商品随步伐轻微晃动并被镜头展示”。不要写拿起、插入、取出。
- `end_state`：最后一帧状态，必须包含商品仍在侧袋、画面中心、清晰可见、没有第二个商品。

## Prompt 块结构
最终给视频模型的 prompt 必须是中文自然语言，不要输出 JSON、评分字段、风险字段或内部槽位名。建议顺序如下：

1. 画幅和风格：竖屏 9:16，写实商品短视频，真实通勤广告质感。
2. 镜头来源边界：这是新的独立分镜，不使用素材图作为第一帧，不接续上一镜头空间。
3. 新场景边界：明确地点、时间、背景只属于新分镜；上一镜中的桌面、房间、手部动作、包装盒不出现。
4. 素材理解转商品复刻：说明“根据素材理解商品外观”，再写可复刻特征；对 logo/文字写低精度处理。
5. 空间布局：商品已经放在背包侧袋里，背包、身体、背景的相对位置清楚。
6. 单一动作：只表现通勤移动和商品随包轻微晃动，不表现放入过程。
7. 0-5 秒时间块：每段只推进同一个连续动作，不跨地点。
8. 接触和物理：商品底部被侧袋托住，侧袋边缘包住商品，不漂浮、不穿模、不跳出。
9. 相机运动：跟拍、轻微推进或收紧，不切换到别的房间。
10. 最后一帧状态：商品稳定在侧袋中，关键外观清晰，单一主商品。
11. 简短边界句：不出现第二个商品、不回到素材场景、不自动生成未描述的大 logo。

## 正例

### 正例 A：通勤水杯，低 logo 风险，可直接填槽
```text
竖屏 9:16，写实商品带货短视频，真实清晨通勤广告质感。这个镜头是新的独立分镜、新地点、新时间，不使用素材图作为第一帧，也不接续上一镜的桌面、室内房间、手拿动作或包装画面。画面发生在{new_scene_description}，背景只属于这个新分镜，不能把素材里的背景物体搬进来。

根据素材理解商品外观：{source_material_observation}。现在要在新场景里用文字复刻同一类商品，商品描述为{product_description}。可复刻特征必须清楚呈现：{product_reconstructable_features}。如果商品上有小 logo、小标签或细小印刷，只表现为低对比度的小面积标识，不要生成大号新 logo，不要编造清晰文字，不要求一字不差。

画面主体是{subject_description}。商品从第一帧开始就已经稳定放在{product_location}里，不是手里拿着，也不是正在塞进去。侧袋边缘包住商品下半部，商品底部被袋底支撑，露出的上半部朝向镜头，轮廓完整可见。背包、衣服、侧袋和商品都属于同一个新地点空间，背景通勤人群保持虚化，不能突然切回素材场景。

镜头在{camera_position}，只表现“{single_action}”这一件事。0.0-1.0 秒：通勤者从画面一侧自然进入，背包侧面占据画面中部，{product_location}和商品一起进入视觉中心；商品已经放好，随步伐只有轻微惯性晃动。1.0-2.2 秒：镜头保持贴近跟拍，侧袋布料或网袋轻微拉伸，商品与袋口有真实接触，不能悬空，不能滑出。2.2-3.4 秒：自然光扫过商品表面，展示主要颜色、材质、盖子、杯身比例、边缘线和素材中可复刻的结构特征；背景继续移动但不换地点。3.4-4.4 秒：镜头轻微收紧到背包侧袋近景，商品仍只有一个，背包上不出现额外同款商品，手也不进入画面取出商品。4.4-5.0 秒：人物步伐放慢，画面稳定在商品近景，最后一帧是{end_state}。整个镜头不出现第二个商品，不跨越到办公室室内，不表现拿起、插入、打开、饮用或取出。
```

### 正例 B：半透明水瓶，强调文字复刻而非首帧绑定
```text
竖屏 9:16，写实通勤生活方式短视频。新的独立分镜，不以前一张素材图作为画面开头，不继承素材里的桌面、灯光、墙面或拍摄角度。场景是{new_scene_description}，这是一个明确的新地点和新时间，画面中所有背景都服务于通勤携带展示。

根据素材理解，商品是{product_description}，核心可复刻特征是{product_reconstructable_features}；其中透明度、颜色、瓶身比例、盖子形状、挂环位置优先保持，瓶身小字或品牌标记只作为模糊小标识处理。商品已位于{product_location}，从第一帧起就在侧袋内，侧袋弹力边缘压住瓶身下半部，瓶身上半部露出，能看见半透明材质和瓶盖结构。画面主体为{subject_description}，镜头从{camera_position}观察。

本镜头只做一个连续动作：{single_action}。0.0-1.0 秒：人物背着包沿同一条路前进，商品随背包进入画面中心，侧袋和瓶身保持贴合。1.0-2.0 秒：镜头与背包保持相同速度，瓶身有轻微摆动，摆动来自步伐惯性，不能自己旋转或跳出侧袋。2.0-3.2 秒：环境光穿过半透明瓶身，显出素材里对应的颜色、厚度、瓶盖和挂环轮廓；不要出现另一个手持水瓶。3.2-4.2 秒：镜头稍微降低到袋口高度，展示袋口包裹、瓶底承托和背包布料纹理，背景只保持同一地点的移动虚化。4.2-5.0 秒：人物在画面边缘停稳，商品仍在{product_location}，最后一帧是{end_state}。不要切换地点，不要把放入侧袋的过程补出来，不要生成大而清晰的虚构 logo。
```

### 正例 C：折叠伞或细长收纳筒，单动作安全版
```text
竖屏 9:16，写实商品展示短视频，雨后通勤氛围。这个分镜是全新的场景重建，不从素材首帧开始，不把素材里的桌面、白底棚拍、手部姿势或包装盒放进同一画面。新场景为{new_scene_description}，地点、光线和背景在 0 到 5 秒内保持一致。

商品来自素材理解，但在视频里用文字复刻：{source_material_observation}。视频中商品写成{product_description}，必须保留的可复刻特征包括{product_reconstructable_features}。若素材中有小标签、细线图案或品牌字样，只保留“小面积低对比度标识”的感觉，不要追求精确文字，不要把 logo 放大成主视觉。

画面主体是{subject_description}，商品已经插在{product_location}，露出的高度和素材中商品的细长比例一致。袋口对商品有包裹和支撑，商品与背包一起运动，不单独漂浮。镜头位置是{camera_position}，只表现{single_action}。0.0-0.8 秒：背包侧面进入画面，商品处在侧袋中，轮廓清楚但背景仍有通勤运动感。0.8-1.8 秒：人物持续向前走，商品随背包轻轻晃动，底部始终在袋内，顶部露出供观众识别。1.8-3.0 秒：镜头平稳靠近，展示商品的主色、材质、端盖或收口结构、表面纹理，不生成额外配件。3.0-4.2 秒：镜头保持同一侧后方角度，背包布料、侧袋弹力边缘和商品接触关系稳定，不能出现手把商品取出。4.2-5.0 秒：人物放慢，商品停在画面中心近景，最后一帧是{end_state}。整段只有一个主商品、一个新地点、一个连续动作。
```

## 反例

### 反例 1：首帧素材和新地点冲突
```text
第一帧必须完全使用素材图里的桌面水杯，然后镜头拉远变成地铁站，水杯已经在背包侧袋里。
```
问题：同一 5 秒镜头里既要求素材首帧，又要求新地点和新承载位置。视频模型会把桌面、地铁站、背包侧袋混成一个空间，或生成第二个水杯。

### 反例 2：纯自由生成导致错误商品
```text
一个上班族背着包走路，侧袋里有一个好看的水杯，镜头展示水杯。
```
问题：没有素材理解，没有颜色、形状、材质、盖子、比例等复刻信息，模型会生成任意杯子，商品身份丢失。

### 反例 3：多个动作塞进 5 秒
```text
手从桌上拿起杯子，打开背包侧袋，把杯子塞进去，背上包走进写字楼，最后坐到工位喝水。
```
问题：拿起、插入、背包移动、换地点、饮用是多个动作，5 秒内会跳切、漏动作或复制商品。

### 反例 4：高精度 logo 商品
```text
黑色杯身上有复杂冠军队徽和三行英文标语，要求侧袋移动时所有字母和队徽完全一致。
```
问题：文字复刻新场景不能保证高精度 logo 与文字，必须改用更强图片锚定策略或避开 logo 展示。

### 反例 5：把上一镜塞进同一空间
```text
延续上一镜厨房台面的水杯，下一秒用户已经走在户外，杯子在背包侧袋里，厨房台面还在背景里。
```
问题：新分镜必须明确新地点、新时间；不要让上一镜空间残留。

## 失败标签
评估失败时优先使用以下标签：
- `wrong_product`：商品颜色、形状、材质或结构与素材理解不符，变成另一类商品。
- `shape_changed`：商品比例或关键结构明显改变，例如直筒杯变成胖罐、翻盖变成吸管杯。
- `logo_changed`：提示词错误要求高精度 logo，或模型生成了夸张、清晰但不属于素材的大 logo。
- `extra_product` / `second_product_generated`：手里、背包另一侧、背景桌面又出现同款或相似商品。
- `scene_conflict`：新地点里混入素材首帧空间、上一镜房间、桌面或白底棚拍。
- `bad_physics`：商品悬空、穿过侧袋、底部无支撑、从袋中滑出又自动回位。
- `prompt_intent_lost`：镜头重点变成通勤者、城市风景或背包整体，商品不清晰。
- `action_not_completed`：没有完成侧袋通勤展示，或者动作变成拿取、打开、饮用。

## 校验规则
composer 输出最终 prompt 前必须逐条检查：
- 是否明确写了“新的独立分镜、新地点、新时间，不使用素材图作为第一帧，不接续上一镜空间”。
- 是否把 `source_material_observation` 转换成商品复刻语言，而不是让视频模型自己“看参考图里的商品”。
- 是否写清 `product_reconstructable_features`，且至少包含颜色、整体形状、材质、关键结构、尺寸比例五类信息中的三类。
- 是否明确商品从第一帧开始已经在 `product_location`，没有写“拿起”“塞入”“取出”“打开”“饮用”。
- 是否只有一个动作，且 0-5 秒时间块都发生在同一新地点。
- 是否有物理接触描述：侧袋边缘包裹、底部承托、随背包轻微晃动、不漂浮。
- 是否避免高精度 logo 要求：只能写小面积、低对比度、模糊标识，不能写清晰文字完全一致。
- 是否明确最后一帧 `end_state`，并包含商品仍在侧袋、单一商品、清晰可见。
- 是否没有把内部评分、风险分、槽位名、JSON、校验规则塞进最终给视频模型的 prompt。

## 给 LLM 的决策提示
- 如果用户既给素材又要新地点，先判断是否必须首帧绑定。只要新地点与素材场景不同，就不要写“第一帧就是素材图”；应写“根据素材理解商品外观，在新场景中复刻”。
- 如果商品身份主要靠 logo、文字、包装插画、IP 图案识别，不要使用本 skill。选择更强锚定方案，或把镜头改成避开 logo 的低风险角度。
- 如果商品是普通瓶、杯、伞、收纳筒，且颜色形状材质可描述，本 skill 适合。描述越具体，商品越不容易变形。
- 如果分镜想讲完整过程，把动作拆成多个分镜：原场景拿起、侧袋放置、通勤展示分别处理。本 skill 只负责“已经在侧袋中通勤展示”。
- 如果有多张素材图，只抽取对同一商品外观一致的特征；不要要求视频里同时出现每张素材的角度、背景和手势。
